# PromptLab 使用手册

> 读完这篇，不懂 AI 也能用 PromptLab

---

## 一、这是什么

**打个比方：**

你给 AI 助手（比如 ChatGPT）发指令，让它帮你干活。这个指令就叫 **Prompt（提示词）**。

现在的问题是什么？你的 Prompt 散落在聊天记录里、文档里、脑子里，改了也不知道改好改坏。

PromptLab 就是帮你**统一管理这些 Prompt 的工具**，像程序员用 Git 管代码一样。

---

## 二、能干嘛

| 功能 | 用大白话说 |
|------|------|
| 📝 存 Prompt | 一个地方存你所有的 Prompt，不会丢 |
| 📋 看历史 | 改了 Prompt 能看改了什么，改坏了能退回 |
| 🆚 对比测试 | 两个版本同时跑，看哪个效果更好 |
| 📊 看分数 | 每次测试给出分数，数据说了算 |
| 🌐 中英文 | 默认中文，右上角可以切英文 |

## 三、怎么用

### 第一步：打开网站

浏览器打开 **https://promptlab-ten-blond.vercel.app**

你会看到首页——往下翻，有三张卡片告诉你它能做什么。

### 第二步：创建项目

1. 点击 **项目**（顶部导航条）
2. 点 **创建项目** 按钮
3. 起个名字，比如"客服机器人"
4. 点确定

一个 "项目" = 一个用途。比如"客服机器人""小红书文案""代码助手"各是一个项目。

### 第三步：写 Prompt

1. 点进你刚创建的项目
2. 点 **新建提示词**
3. 在弹出的编辑框里写你的 Prompt

**什么是 Prompt？** 就是你给 AI 下的指令。举个例子：

```
你是一个{{角色}}，请用{{语气}}的语气回答：{{问题}}
```

其中 `{{角色}}`、`{{语气}}`、`{{问题}}` 是**变量**——用的时候替换成具体文字。比如：角色=客服、语气=友好、问题=怎么退款。

### 第四步：改一个版本

1. 点右上角 **编辑**
2. 改一下文字，比如把"友好"改成"简洁专业"
3. 点 **保存修改**

这时候你就有两个版本了：v1（友好版）和 v2（简洁专业版）。

### 第五步：对比哪个好

1. 点 **新建实验**
2. 系统自动用 AI 跑两个版本，分别打分
3. 看结果——比如 "v1 得分 72，v2 得分 89，v2 胜出"

**以前：** "我觉得 v2 好"（猜的）
**现在：** "数据说 v2 好 17 分"（确定的）

## 四、常见问题

### Q：我不是程序员，能用吗？

能。你只需要会用浏览器，会打字。

### Q：要花钱吗？

线上版完全免费，API Key 已帮你配好，直接就能用。

如果你自己下载部署，才需要去 DeepSeek 或通义千问申请免费 API Key（开发测试不花钱）。

### Q：为什么看不到"新建实验"按钮？

因为一个 Prompt 至少要有 2 个版本才能对比测试。先去改一版 Prompt，保存后就有"新建实验"按钮了。

### Q：我的 Prompt 存在哪？

存在你自己的电脑或服务器上。不会发到任何第三方。

### Q：为什么叫 PromptLab？

Prompt = 提示词，Lab = 实验室。合起来就是"提示词实验室"——做实验看看哪个 Prompt 更好。
