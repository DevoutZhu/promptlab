# PromptLab 使用手册

> 当前版本：v0.1 MVP，本地开发模式，数据为演示数据

---

## 怎么打开

在终端运行（我已经帮你启动了）：

```bash
cd D:\my-project\promptlab
pnpm dev
```

浏览器打开 **http://localhost:3000**

---

## 一、首页

打开后看到的第一个页面。从上到下：

1. **顶部大标题**："Manage Prompts Like Code" + 两个按钮
2. **三个功能卡片**：版本控制 / A/B 测试 / 团队协作
3. **三步流程图**：接入 LLM → 写测试 → 发布
4. **底部 CTA**："开始使用"

目前都是展示页面，按钮跳转到 Dashboard。

---

## 二、项目列表页

地址：**http://localhost:3000/dashboard**

这个页面展示：
- 6 个演示项目的卡片（Customer Support Bot、Content Generator 等）
- 顶部统计：总项目数 / Prompt 总数 / 活跃实验 / 平均分
- 搜索框可以筛选项目
- 点击任意项目卡片进入项目详情

---

## 三、项目详情页

地址：**http://localhost:3000/dashboard/1**（数字是项目 ID）

这个页面展示：
- **统计卡片**：该项目下有多少 Prompt、实验、平均分
- **Prompt 列表表格**：名称、版本号、更新时间、评分
- **右侧实验列表**：正在运行的 A/B 实验
- **评分趋势图**：30 天的分数变化折线图
- 点击表格里的 Prompt 名称，进入 Prompt 详情

---

## 四、Prompt 详情页（核心页面）

地址：**http://localhost:3000/dashboard/1/prompts/1**

这是最重要的页面，包含：

### 4.1 Prompt 编辑器
- 中间大文本框，显示 Prompt 模板内容
- `{{变量名}}` 会显示为蓝色标签
- 点击底部变量栏可以插入变量
- 右上角可以切换只读/编辑模式

### 4.2 版本切换
- 顶部下拉框可以切换不同版本
- 右侧时间线显示 v1 → v2 → v3 的历史

### 4.3 Diff 对比
- 选择两个版本，左右分栏对比差异
- 新增内容标绿，删除内容标红

### 4.4 评测结果
- 大字显示当前版本评分
- 通过/失败用例数
